# GitTrix

## Initialize a repo:

```git init .```

 ## Check the Status for changes and such
```git status```

## Add a file(s) to the staging area

One file: ```git add .FILENAME```

All them files: ```git add .``` 

## Commit 'em

```git commit -m "-m is for comment"```

## Check commit history

```git log```

## Push the changes up to Github
```git push```
